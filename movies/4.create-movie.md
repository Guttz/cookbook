# Create a movie

## Story

As an API client I want to create a new movie resource

## API behaviour

```http
GET https://hydra-movies.herokuapp.com/
```

Response body (excerpt):

```json
{
  "@id": "/",
  "collection": {
    "@id": "movies",
    "@type": [
      "Collection",
      "mov:Movies"
    ]
  }
}
```

Api documentation (excerpt):

```json
{
  "@id": "/doc",
  "supportedClass": [
    {
      "@id": "mov:Movies",
      "@type": "Class",
      "supportedOperation": {
        "@id": "mov:create-movie",
        "@type": [
          "Operation",
          "schema:CreateAction"
        ],
        "title": "Create movie",
        "description": "Creates a new movie",
        "method": "POST",
        "expects": "schema:Movie",
        "returns": "schema:Movie"
      }
    },
    {
      "@id": "schema:Movie",
      "@type": "Class",
      "title": "Movie",
      "description": "A single movie",
      "supportedProperty": [
        {
          "property": "schema:name",
          "title": "name",
          "description": "The movie's name"
        },
        {
          "property": "schema:description",

        },
        {
          "property": "schema:duration",

  ]
}
```

```http
POST https://hydra-movies.herokuapp.com/

{
```
